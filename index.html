Codekit demo page

The following are instructions on how to set up this project in Codekit. For more tutorials see the Codekit video page.

  1. Open Codekit
  2. Drag your project folder into Code Kit. Be sure to drop into the Project section rather than the framework section.
  3. Sass setup:
    1. Project > languages > Choose Sass
    2. Output Style: Compact or compress
    3. The Output Folder: "To this folder" and change the path/name to ../dist
    4. Change Name The Output File to *-min.css
  4. Javascript setup:
    1. Project > languages > Choose Javascript
    2. Check Syntax With: JSHint
    3. Output Style: Minified
    4. The Output Folder: "To this folder" and change the path/name to ../dist
    5. Leave Name The Output File at *-min.js
  5. Syntax checkers > JSHint: Environments > devel
  6. Select preview to start the local server
